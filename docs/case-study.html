Methodology and Data Processing

Our comprehensive analysis followed these key steps:

  • Adjusted all costs to 2018 dollars for fair comparison
  • Combined data from multiple authoritative sources
  • Implemented rigorous data validation procedures
  • Applied statistical testing for all relationships
  • Used a systematic approach for handling missing data:
    • Primary: Used exact values when available
    • Secondary: Applied temporal averaging within states
    • Tertiary: Utilized neighboring states' data (especially for IN and NM)
    • Final: Applied national averages where necessary
  • Calculated key metrics including cost burden and participation rates
  • Ensured consistent visualization standards throughout

Data Quality Considerations

Our analysis addressed several data challenges:

  • The original dataset contained significant missing data in key childcare columns (31.7% missing for center-based care)
  • States like Indiana and New Mexico required special handling due to missing data
  • For Indiana, we utilized data from neighboring states including Illinois ($2,920), Kentucky ($1,530), Michigan ($1,897), and Ohio ($1,857)
  • For New Mexico, we incorporated data from Arizona ($1,900), Colorado ($2,713), Oklahoma ($1,999), and Texas ($1,310)
  • Our careful imputation process ensured these states were properly represented while maintaining analytical integrity
