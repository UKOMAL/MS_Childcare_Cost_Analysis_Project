Troubleshooting

If you encounter issues with the dashboard:

  • Try the JavaScript Test Page to verify script loading
  • Check your browser console for error messages (F12 or right-click > Inspect > Console)
  • Try clearing your browser cache or using incognito/private mode
  • For local development, use a local server to avoid CORS issues
